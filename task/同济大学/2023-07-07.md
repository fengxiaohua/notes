搭建gitlab-runner

没有日志输出 升级gitlab-runner版本解决

安装gitlab-runner
sudo curl -L "https://packages.gitlab.com/install/repositories/runner/gitlab-runner/script.rpm.sh" | sudo bash
sudo yum install gitlab-runner
gitlab-runner register
